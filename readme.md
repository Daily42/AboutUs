# 🗓️ Daily42 🗓️

_42에서의 시간을 더욱 가치있게!_

Daily42는 42서울과 연동되는 장소 리스트를 기반으로 한 일정 등록과 언제, 어디서, 얼마동안 무슨 일이 벌어지는지 정확히 알 수 있는 일정 관리 기능을 제공하는 사이트입니다.

## 기술 스택

#### 🌐 Frontend
- `React`
- `TypeScript`

#### 🖥️ Backend
- `Nest.js`
- `MySQL`
- `TypeORM`
